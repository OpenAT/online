REST API/OpenAPI/Swagger UI

Host Swagger UI for displaying API documentation.

Creates a public endpoint to serve the official Swagger UI. Index HTML is modified via a GET parameter api in order to directly show documentation for a given API.

Example

<base address>/api/swagger-ui?api=http%3A%2F%2Fserver.com%2Fapi%2Fv1%2Fswagger.json

Security
  • API URLs are checked to be valid urls
  • API URL must match the controller request
    • Means external APIs will be refused by default
