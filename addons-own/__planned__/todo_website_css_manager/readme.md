# website_css_manager
This Manager holds a list of custom css or paths to css files together with information where and how
this custom css can be used.

## PLANNED FIELDS:
- name
- css
- css_file
- Load in Assets Frontend
- Load in Assets Backend
- Load in Assets Editor
