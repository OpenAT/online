Description

Provides static assets and routes for Swagger UI and Swagger Editor.

Usage

Simply call the desired endpoint for only the UI or the Editor. Via GET parameter, send the URL to the specification file to be displayed.

Examples

  • /swagger-ui/index.html?swagger_spec_url=(URL encoded URL to spec file)
  • /swagger-editor/index.html?swagger_spec_url=(URL encoded URL to spec file)
  • Without a spec URL, a static pet store demo asset is displayed.

Updating Swagger libraries

Locally download new Swagger packages via npm. Use the dist-packages, e.g. swagger-ui-dist or swagger-editor-dist.

Within node_modules rename the downloaded package folder to include the version number, e.g. swagger-ui-dist_x_xx_x

Copy the new folder into the module under static/src/lib. Adjust the symbolic link, to point to the new version, e.g. have the link swagger-ui-dist point to swagger-ui-dist_x_xx_x.
